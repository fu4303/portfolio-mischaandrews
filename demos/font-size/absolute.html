Absolute sizing

This page doesn’t change.

This page is built mostly with absolute sizing (in this case, with ‘px’ sizing).

When you change your browser’s font size setting, the layout and text will stay the same size (even though it’s otherwise responsive).

Try it out by adjusting your browser settings (it’s not the same as zooming). In Chrome, it’s under Settings > Advanced > Web content > Font size.

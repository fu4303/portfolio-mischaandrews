Mixed sizing

This page behaves inconsistently.

This page is built using a mix of absolute and relative sizing (in this case, some ‘px’ and some ‘em’ sizing).

When you change your browser’s font size setting, the font and layout sizing will behave inconsistently (even though it’s otherwise responsive). With more complex pages, this can break the layout.

Try it out by adjusting your browser settings (it’s not the same as zooming). In Chrome, it’s under Settings > Advanced > Web content > Font size.
